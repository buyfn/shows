# TV shows

This is an SPA displaying TV shows and episodes info.
Demo: https://frosty-roentgen-6811e0.netlify.com/

## How to run

### Install dependencies
```
npm install
```

### Run app
```
npm start
```

### Open in browser
Open `http://localhost:3000` in browser

Random show: `http://localhost:3000/show/random`
